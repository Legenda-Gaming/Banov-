RaG Baseitems instructions and how-to:

The following text is already formatted for Discord, if you want to make a dedicated channel for this.





__**Normal Crafting**__

**Small Metal Sheets(needed for Workbench)**

```
Hacksaw + Metal Plate = 60 Small Metal Sheets / Be aware that you will need Sharpening stones as it will ruin the Hacksaw really fast!
```

**Small Plastic Sheets(needed for Workbench)**

```
Pliers + Water Bottle = 10 Small Plastic Sheets
```




__**Workbench Crafting**__

*Everything gets crafted with a Hammer! You will always see the desired result as a preview before you craft it!*

**Crates**

*There are 3 different "Categories" of crates. Small, Medium and Large which need different amounts of the following:*

```
Requirements to craft:

- Wooden Planks ( Small:4 // Medium: 8 // Large: 16 )
- Small Metal Sheets ( Small:20 // Medium: 40 // Large: 80 )
- Nails ( Small:20 // Medium: 40 // Large: 80 )
```

**Metal Ammo Boxes**

*There are 3 different Colours to craft. Red - Green - Black*

```
Requirements to craft:

- Small Metal Sheets - 60
```

**Plants**

*There are 6 different Plants to craft.*

Requirements to craft:
